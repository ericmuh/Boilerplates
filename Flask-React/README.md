# Flask with React Frontend
simple application with React front end and Flask Backend


## Requirements
* Python3.x
* pip3
* Nodejs-10.x 



## Installation
```bash
pip install -r requirements.txt
npm install

```

##usage
Run two terminals 
one running npm 
```bash
npm start
```

The other running flask
```bash
cd Backend
python3 server.py
```
