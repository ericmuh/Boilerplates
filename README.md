# Flask with React Frontend
simple application with React front end and Flask Backend


## Requirements
* Python3.x
* pip3
* Nodejs-16.x 



## Installation
```bash
pip install -r requirements.txt
npm install

```

##usage
Run two terminals 
one running npm/yarn and the other for the pytho flask

***using yarn
```bash
cd Frontend
yarn dev 
```
***using npm
```bash
npm run dev
```

The other running flask
```bash
cd Backend
python server.py
```
